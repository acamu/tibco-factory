Code Review plugin for TIBCO BusinessWorks

Automatic TIBCO BusinessWorks code review

FCCodeReview is an automatic tool to generate Code Reviews of TIBCO BusinessWorks projects.

The rules applied to perform a Code Review can be:

  • the built-in rules, written by FastConnect based on common good practices
  • custom rules, which can be written within a simple TIBCO BusinessWorks project
