code-review:prepare-code-review

Full name:

fr.fastconnect.factory.tibco.bw.codereview:code-review-plugin:2.0.0-SNAPSHOT:prepare-code-review

Description:

This goal will prepare the target project for the Code Review.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-resources.
