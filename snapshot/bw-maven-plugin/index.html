Maven plugin for TIBCO BusinessWorks

Enable Java-like lifecycle management in TIBCO BusinessWorks

Advanced Configuration Management

An advanced configuration management is introduced during all the application lifecycle, from development to deployment through packaging.

TIBCO Designer aliases

BWMaven will automatically adds the aliases in TIBCO Designer based on dependencies declared in the POM file of the project.
