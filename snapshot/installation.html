FastConnect TIBCO Factory

A Factory, not a framework.

Installation

Three steps are required to install a plugin of the FC TIBCO Factory:

  • ensure that your system meets the required prerequisites listed on this page.

  • setup a basic configuration for your projects and environment. All plugins of the FC TIBCO Factory share this common configuration.

  • installation is then performed by creating (or updating) POM files for the TIBCO projects. This process is called mavenization.

Prerequisites

  • Internet access (at least Maven central)
  • Maven 3.1+ installed
  • JDK 1.6+ installed
  • a valid TIBCO installation
    • TRA 5.7+
    • BW 5.9+

Maven central

All plugins of the FC TIBCO Factory are provided as standard Maven plugins, all available on the Maven central repository.
